1. O que é um repositório no Git e qual a sua importância?
    É um lugar onde você pode armazenar seu código, seus arquivos, histórico de revisão de cada arquivo e facilitando para outros usuários a mexer no mesmo projeto.

2. Qual a função do Git Bash?
    O Git Bash é um comando que permite gerenciar seu projeto através de uma linha de comando, ou seja, o prompt.

3. O que faz o comando git add e qual a diferença entre git add e git add .?
    O "git add ." envia todos os arquivos para um repositório, já o "git add" envia um único arquivo.
    
4. Por que utilizamos o ponto (.) no comando git add .?
    Para enviar todos os arquivos para o repositório.

5. Explique o comando git commit e por que é importante criar uma mensagem descritiva.
    É usado para salvar um projeto no github, onde se você acessá-lo depois saberá em qual ponto você parou.

6. Qual a função do comando git push?
    Enviar os arquivos para o github.

7. Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
    Caso o projet de algum erro, será possível voltar a versão anterior pelo commit.

8. Explique a função das tags HTML e como elas estruturam uma página.
    Elas permitem definir diferentes tipos de conteúdo, como texto, imagens, links e etc.

9. O que são atributos em uma tag HTML e qual a sua função?
    Atributos em HTML são detalhes extras que você coloca nas tags para ajustar ou personalizar o jeito que um elemento aparece ou funciona na página.

10. O que é o Emmet e como ele facilita o desenvolvimento de páginas HTML?
    Emmet é uma ferramenta que facilita a escrita de código HTML e CSS, onde é possível completar uma tag sem digitar ela toda facilitando assim a digitação do código.

11. O que é o box-sizing no CSS e por que usamos ele no reset inicial?
    Ele define como as dimensões de um elemento, largura e altura, são calculadas.

12. Por que é importante utilizar tags semânticas no HTML?
    Para facilitar a compreensão do usuário.

13. O que são media queries no CSS e qual a sua função?
    Media quieres no CSS permite aplicar estilos diferentes como tamanho da tela, orientação e etc.

14. Qual a função do GitHub Pages?
    É utilizado para colocar o site no ar para conseguir testá-lo em outros aparelhos.

15. O que é o Flexbox no CSS e para que ele é utilizado?
    Flexbox no CSS é um modelo de layout que facilita o alinhamento e distribuição de itens dentro de um container, tanto na direção horizontal quanto vertical.

16. Qual a diferença entre justify-content e align-items em um container Flexbox?
    justify-content alinha os itens na direção principal (horizontal ou vertical).
    align-items alinha os itens na direção perpendicular (vertical ou horizontal).

17. Como o flex-grow funciona e qual o seu propósito no layout Flexbox?
    É uma propriedade do Flexbox que controla como os itens dentro de um container crescem para preencher o espaço sobrando.

18. Explique a diferença entre flex-direction: column; e flex-direction: row;
    flex-direction: row; organiza os itens horizontalmente, da esquerda para a direita.
    flex-direction: column; organiza os itens verticalmente, de cima para baixo.
